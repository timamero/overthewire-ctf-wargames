# OverTheWire CTF Wargames - Walkthroughs & Solutions

A collection of walkthroughs, solutions, and notes for completing OverTheWire's Capture The Flag (CTF) wargames. This repository documents my progression through various security challenges, focusing on developing practical hands-on security skills.

## Overview

[OverTheWire](https://overthewire.org/) provides a series of wargames designed to teach security concepts through practical, interactive challenges. This repository serves as both a personal reference and a portfolio demonstrating my commitment to security fundamentals and continuous learning.

## 🎯 Completed Challenges

### Bandit Wargames ✅

The Bandit wargames are an excellent introduction to the Linux command line, user interactions, and fundamental security concepts. I've successfully completed all 33 levels of the Bandit wargame series.

**Skills Gained:**

- **Linux Command Line Mastery**
  - File navigation, manipulation, and analysis
  - Advanced grep, sed, and awk usage
  - File permissions and ownership
  - Process management and monitoring

- **Security Fundamentals**
  - SSH key-based authentication
  - Password handling and secure credential storage
  - Binary file analysis and exploitation
  - Understanding vulnerability chains

- **Scripting & Automation**
  - Bash scripting for problem-solving
  - Creating automated solutions for repetitive tasks
  - Understanding shell special variables and operators
  - Logic-based programming in a Linux environment

- **Debugging & Problem Solving**
  - Reading and interpreting complex output
  - Identifying and exploiting common vulnerabilities
  - Reverse engineering techniques
  - Data decoding (base64, hexadecimal, etc.)
